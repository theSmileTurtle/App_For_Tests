# App_For_Tests
Для достижения решения поставленной задачи данной курсовой работы был выбран для реализации объектно-ориентированный подход. Язык программирования, на котором велась разработка – это C#. В качестве среды разработки была выбрана программа Visual Studio 2022.
Так как программа работает с созданием и редактированием вопросов, были разработаны классы для каждого типа вопросов (базовый класс Question и производные от него SingleAnswer, MultiAnswer, SentenceAnswer, CompareAnswer, InsertAnswer).
Так же для обработки результатов ответов на вопросы, был создан вспомогательный класс EndAnswer.
Для сохранения данных о тесте была разработана структура хранения данных в XML-файле. В данную структуру входит модуль для хранения характеристик оценивания и модуля для хранения каждого вопроса по отдельности.

После запуска программы открывается окно авторизации, в этом окне можно выбрать в каком режиме продолжить работу.

**Режим "Преподаватель"**
После выбора режима преподавателя открывается окно создания теста. В данном окне есть элементы для создания вопроса, выбранного типа.
Для вопросов типа Simple отображаются поля для вариантов ответов и элементы, позволяющие выбрать только один из них правильным.
Для вопросов типа Multi отображаются поля для вариантов ответов и элементы, позволяющие выбрать несколько из них правильными.
Для вопросов типа Sentence отображается только одно поле для ответа. В это поле можно ввести слово или словосочетание.
Для вопросов типа Compare отображаются поля для вариантов ответов, поля для вариантов соотношения и элементы, позволяющие выбрать какой элемент какому соответствует.
Для вопросов типа Insert становятся активными кнопки добавления текста и ответа. Которые позволяют добавить нужные поля для редактирования вопроса.
После нажатия на кнопку «Завершить» открывается окно выбора параметров оценивания, и диалоговое окно для сохранения теста в файл.


**Режим "Студент"**
После выбора режима студента открывается стандартное диалоговое окно для выбора файла, после открывается окно решения теста. В данном окне есть элементы для выбора ответа на вопрос, соответствующего типа.
После нажатия кнопки «Завершить тестирование» открывается окно с результатами тестирования. В этом окне указывается полученная оценка, количество правильных ответов и их процент.
